== Description ==

Music Tutorials by Tuner Coin Plugin simply adds a customizable widget which displays the latest news and tutorials for musicians.All the posts are syndicated from the official Tuner coin Blog. This mainly inludes tutorials to set up studios for professional as well as home recording and music production. 

More Information at <a href="http://www.tuner.co.in/">Official Site</a>

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==
Simplest Way
1. Go to You installed plugins in your admin dashboard.
2. Click on "Add New Plugin" on the top.
3. Search for "Music Tutorials By Tuner Coin" in WordPress Plugins Search.
4. Install and activate it.

Simple Way
1. Go to WordPress.org Plugins directory.
2. search for "Music Tutorials By Tuner Coin" and then download it.
3. Go to your installed plugins and then upload it.
4. Activate the plugin.


Hard Old Way
1. Extract `music-tutorials-by-tuner-coin.zip`.
2. Upload the `Tuner Coin-rss-plugin`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
